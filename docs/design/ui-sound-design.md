**布局说明：**

## 2. 配色方案

### 2.1 主色调

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 主蓝色 | `#3B82F6` | 标题、按钮、选中高亮 |
| 深蓝 | `#1E40AF` | 边框、强调色 |
| 浅蓝 | `#93C5FD` | 背景渐变、禁用状态 |

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 主红色 | `#EF4444` | 危险、警告、重要提示 |
| 深红 | `#991B1B` | 严重警告 |

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 主绿色 | `#22C55E` | 生命值正常、成功 |
| 深绿 | `#15803D` | 确认按钮 |

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 主黄色 | `#EAB308` | 警告、HP 偏低 |
| 金色 | `#F59E0B` | 稀有物品、金币 |

### 2.2 中性色

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 背景黑 | `#1F2937` | 主背景、对话框背景 |
| 深灰 | `#374151` | 次要背景 |
| 中灰 | `#6B7280` | 边框、分隔线 |
| 浅灰 | `#9CA3AF` | 次要文本 |
| 文本白 | `#F9FAFB` | 主要文本 |

### 2.3 UI 元素颜色

**按钮：**
- 普通按钮：背景 `#3B82F6`，文字 `#FFFFFF`
- 确认按钮：背景 `#22C55E`，文字 `#FFFFFF`
- 取消按钮：背景 `#6B7280`，文字 `#FFFFFF`
- 危险按钮：背景 `#EF4444`，文字 `#FFFFFF`
- 悬停/选中：亮度 +15%
- 禁用：背景 `#9CA3AF`，文字 `#6B7280`

**对话框：**
- 背景：`#1F2937` (95% 不透明度)
- 边框：`#3B82F6` (3px)
- 文字：`#F9FAFB`

**菜单选中高亮：**
- 背景：`#3B82F6` (40% 不透明度)
- 边框：`#60A5FA` (2px)

### 2.4 HP 条颜色

| 状态 | 百分比 | 颜色 | 渐变 |
|------|--------|------|------|
| 正常 | > 50% | 绿色 | `#22C55E` → `#16A34A` |
| 警告 | 25% ~ 50% | 黄色 | `#EAB308` → `#CA8A04` |
| 危险 | ≤ 25% | 红色 | `#EF4444` → `#DC2626` |

**PP 条颜色：**
- 正常：蓝色 `#3B82F6` → `#2563EB`
- 偏低 (< 25%)：紫色 `#8B5CF6` → `#7C3AED`

## 3. 字体方案

### 3.1 推荐字体

**像素风格字体（首选）：**
- `Press Start 2P` (Google Fonts)
- `VT323` (Google Fonts)
- `Courier New` (后备)

**无衬线字体（备选）：**
- `Noto Sans SC` (中文)
- `PingFang SC` (macOS)
- `Microsoft YaHei` (Windows)

### 3.2 字号定义

| 名称 | 字号 | 用途 |
|------|------|------|
| 超大号 | 32px - 48px | 游戏标题 |
| 大号 | 24px - 28px | 界面标题、HP/PP 数值 |
| 中号 | 18px - 20px | 菜单选项、对话框正文 |
| 小号 | 14px - 16px | 操作提示、版权信息 |
| 极小号 | 12px | 状态标签、辅助信息 |

### 3.3 字体样式

**标题文字：**
- 粗体
- 可选：文字阴影、发光效果

**正文文字：**
- 常规字重
- 行高：1.5 - 1.6

**选中高亮：**
- 文字颜色：白色或浅黄
- 可添加下划线或外发光

## 附录：音效技术参数

### Web Audio API 实现建议

1. **音效格式**：优先使用 Web Audio API 生成基础音效，后备使用 MP3/OGG
2. **采样率**：44.1kHz
3. **位深**：16-bit
4. **声道**：单声道或立体声
5. **音量**：
   - BGM：-10dB 至 -6dB
   - SFX：-6dB 至 0dB
   - 主音量：可调节 0% - 100%

### 音效分类优先级

1. **必须实现**（核心体验）：
   - 所有 UI 操作音效
   - 战斗基础音效（开始、命中、miss、胜利、失败）
   - 获得物品/金钱音效
   - 存档/读档音效

2. **建议实现**（增强体验）：
   - 各属性技能音效
   - 升级音效
   - BGM（标题、战斗、城镇、道路）

3. **可选实现**（完美体验）：
   - 环境音
   - 更多变体音效
