## 3. 定价平衡说明

### 3.1 价格与效果的关系

#### 消耗品定价公式
```
价格 = 基础价格 × 效果倍率 × 稀有度系数
```

- **低级 HP 恢复**: 每 10 HP ≈ 10 金币
- **中级 HP 恢复**: 每 10 HP ≈ 20 金币（便利加成）
- **高级 HP 恢复**: 每 10 HP ≈ 25 金币（高等级便利加成）
- **状态恢复**: 单一状态 ≈ 100 金币，全状态 ≈ 600 金币
- **复活**: 半复活 ≈ 1500 金币，全复活 ≈ 3000 金币

#### 装备品定价公式
```
价格 = (Σ属性加成值 × 每点价格) × 稀有度系数 × 套装加成
```

- **攻击力/防御力**: 每点 ≈ 100 金币
- **特攻/特防/速度**: 每点 ≈ 100 金币
- **最大 HP**: 每点 ≈ 50 金币
- **百分比加成（暴击/经验）**: 每 1% ≈ 300 金币

#### 稀有度系数
| 稀有度 | 系数 | 说明 |
|--------|------|------|
| 普通 | 1.0x | 基础物品 |
| 稀有 | 2.0x | 效果增强或特殊 |
| 史诗 | 4.0x | 强力效果，通常带特效 |
| 传说 | 8.0x | 顶级物品，极其稀有 |

### 3.2 前期、中期、后期物品价格梯度

#### 前期（新手村，玩家等级 1-15）
- **资金范围**: 0 - 3000 金币
- **主力消耗品**: 低级药水 (50)、解毒药 (80)、普通怪兽球 (100)
- **主力装备**: 木剑 (200)、布甲 (180)、布帽 (100)
- **单件上限**: 1000 金币

#### 中期（城市，玩家等级 15-35）
- **资金范围**: 3000 - 20000 金币
- **主力消耗品**: 中级药水 (200)、全状态恢复药 (600)、高级怪兽球 (300)
- **主力装备**: 铁剑 (800)、铁甲 (1800)、力量戒指 (1000)
- **单件上限**: 5000 金币

#### 后期（大城市，玩家等级 35+）
- **资金范围**: 20000+ 金币
- **主力消耗品**: 高级药水 (500)、复活药 (1500)、超级怪兽球 (600)
- **主力装备**: 钢剑 (2000)、钢甲 (4000)、灵魂宝珠 (8000)
- **单件上限**: 25000 金币

### 3.3 装备价格与属性加成的关系

| 属性类型 | 每点价格（金币） | 示例 |
|----------|------------------|------|
| 攻击力 (ATK) | 100 | +10 ATK = 1000 金币 |
| 防御力 (DEF) | 100 | +10 DEF = 1000 金币 |
| 特攻 (SpATK) | 100 | +10 SpATK = 1000 金币 |
| 特防 (SpDEF) | 100 | +10 SpDEF = 1000 金币 |

## 5. 背包系统设计

### 5.1 物品分类

背包支持以下分类筛选：

| 分类 | 说明 | 包含物品类型 |
|------|------|--------------|
| **全部** | 显示所有物品 | 所有类型 |
| **消耗品** | 可使用的消耗道具 | 药水、状态恢复药、复活药、捕捉球等 |
| **装备** | 可穿戴的装备 | 武器、防具、头盔、靴子、饰品 |
| **关键道具** | 剧情相关重要物品 | 徽章、秘传机、钥匙等 |

### 5.2 堆叠规则

#### 可堆叠物品（消耗品类）
- **堆叠上限**: 99 个/格
- **包含**: 所有消耗品、捕捉球

#### 不可堆叠物品（装备/关键道具）
- **堆叠上限**: 1 个/格
- **包含**: 所有装备、关键道具

#### 堆叠规则示例
```
物品格数据结构:
{
  itemId: "potion",
  quantity: 50,      // 1-99 对于可堆叠物品
  equipped: false,    // 仅装备品有效
  slot: null          // 装备位置，如已装备
}
```

### 5.3 排序选项

背包支持以下排序方式：

| 排序方式 | 说明 |
|----------|------|
| **按名称** | 按物品名称拼音/字母顺序排序 |
| **按价格** | 按出售价格从高到低排序 |
| **按稀有度** | 传说 > 史诗 > 稀有 > 普通 |
| **按类型** | 按物品类型分组排序 |
| **按获取顺序** | 按获得时间先后排序（默认） |

### 5.4 背包操作

#### 基本操作
1. **查看详情**: 选中物品显示详细信息
2. **使用**: 对消耗品和关键道具可用
3. **装备/卸下**: 对装备品可用
4. **出售**: 打开商店时可出售
5. **移动**: 调整物品位置（可选功能）
6. **丢弃**: 销毁物品（关键道具不可丢弃）

#### 装备槽位
```javascript
equipment: {
  weapon: null,       // 武器
  armor: null,        // 防具
  helmet: null,       // 头盔
  boots: null,        // 靴子
  accessory: null     // 饰品（可装备多个？待讨论）
}
```

### 5.5 背包容量

- **基础容量**: 50 格
- **扩充方式**: 通过关键道具或商店购买扩容
- **最大容量**: 100 格
- **扩容价格**:
  - 第一次扩容（50→70）: 5000 金币
  - 第二次扩容（70→100）: 15000 金币
